# 🚗 AI-Integrated Dashcam System

An AI-powered dashcam system designed to analyse real-time video streams and detect potential hazards using computer vision and machine learning.

---

## 🔍 Features

* Real-time video processing using OpenCV
* Object detection (vehicles, pedestrians)
* Hazard detection and alert system (in progress)
* Optimised for low-latency performance

---

## 🛠️ Tech Stack

* Python
* OpenCV
* TensorFlow / PyTorch
* NumPy

---

## 🚀 Installation

```bash
git clone https://github.com/yourusername/ai-dashcam-system.git
cd ai-dashcam-system
pip install -r requirements.txt
python src/main.py
```
